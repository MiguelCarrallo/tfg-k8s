# Kubernetes Manifests

This directory contains the Kubernetes resource definitions used to deploy the project infrastructure.

## Structure

- **apps/** – example applications deployed in the cluster
- **ingress/** – ingress resources exposing services externally
- **network/** – networking test workloads
- **observabilidad/** – monitoring configuration (Prometheus, Grafana)
- **rbac/** – role-based access control definitions
- **storage/** – persistent storage resources
